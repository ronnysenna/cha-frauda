# 🍼 Chá de Fraldas - Frontend Next.js

Frontend profissional e moderno desenvolvido com **Next.js 14**, **TypeScript**, **Tailwind CSS** e **shadcn/ui**.

## 📋 Características

✅ **Responsivo** - Funciona em desktop, tablet e mobile  
✅ **TypeScript** - Type-safe  
✅ **Tailwind CSS** - Styling moderno  
✅ **shadcn/ui** - Componentes profissionais  
✅ **API Integration** - Comunicação com backend  
✅ **Admin Dashboard** - Gerenciamento com PIN  
✅ **Página Pública** - Visualização em tempo real  

## 🔧 Começar

```bash
cd frontend
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## 🔌 API Backend

Configure a URL em `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3000/api
```

Para produção, use `.env.production`:

```
NEXT_PUBLIC_API_URL=https://easypanel.ronnysenna.com.br/api
```

## 📄 Páginas

- `/` - Formulário de registro
- `/admin` - Dashboard (PIN: 1234)
- `/registros` - Registros públicos
